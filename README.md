# Markdoc Docs

## Get started

1. Start the development server:
   ```shell
   npm run dev
   # or
   yarn dev
   ```
2. Open [http://localhost:3000](http://localhost:3000) to view the Markdoc site.
3. Try editing `pages/index.md`. The page auto-updates when you edit and save.
